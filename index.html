Sender.net API Setup Guide

To enable automatic payslip emails, follow these steps to connect your Sender.net account:

  1. Log in to your Sender.net dashboard.
  2. Navigate to Settings > API access tokens.
  3. Click the button to generate a new token. You will receive a long string starting with eyJ...
  4. Copy that exact token and paste it into the "API Access Token" field in the Settings module here.
  5. Enter the verified From Email address you use in Sender.net (e.g. hr@yourdomain.com).
Note: Your token acts as a master key to your Sender account. This script securely uses it to call the https://api.sender.net/v2/messages endpoint to dispatch emails directly to your staff.
